# 🐾 chuku.memories

**chuku.memories** is a personal digital repository and a tribute to a group of friends who have shared the journey from the classrooms of IIFM to the depths of the forest. Named after **Chukka**, the dog we raised together, this project serves as a "Memory Vault" where every shared moment has a home.


## 🌟 The Vision
The website is designed to be a heartfelt journey. 
* **The Entrance:** A dedicated landing page featuring Chukka's logo and a personal letter to my friends.
* **The Worlds:** Four unique, interactive portals. Each icon leads to a dedicated "world" (timeline) for a specific friend, spanning from the very first memory to the present.
* **Multimedia:** A mix of high-definition photos and videos hosted via Google Drive.

## 🛠️ Built With
* **HTML5:** For the structural skeleton and semantic layout.
* **CSS3:** For the "Forest & Flora" aesthetic, featuring a responsive timeline and circular icon design.
* **Google Drive API (Direct Linking):** Used as a cloud storage solution for images and videos to keep the repository lightweight.
